Here’s the **fastest, modern way** to create a React project using Vite with the name **`star-wars-01`** 🚀

---

# ⚡ 1. Run the create command

In your terminal:

```bash
npm create vite@latest star-wars-01
```

---

# 🧭 2. Choose options

You’ll be prompted:

* **Framework:** → `React`
* **Variant:** → `JavaScript` (or `TypeScript` if you prefer)

---

# 📦 3. Go into the project

```bash
cd star-wars-01
```

---

# 📥 4. Install dependencies

```bash
npm install
```

---

# ▶️ 5. Start the development server

```bash
npm run dev
```

---

# 🌐 6. Open in browser

You’ll see something like:

```
http://localhost:5173/
```

Open it → your React app is running 🎉

---

# 🧱 What you just created

Your project structure will look roughly like:

```text
star-wars-01
 ├── index.html
 ├── package.json
 ├── src/
 │    ├── App.jsx
 │    ├── main.jsx
 │    └── assets/
 └── vite.config.js
```

---

# 🧠 Important files (quick understanding)

* **`main.jsx`** → entry point (bootstraps React)
* **`App.jsx`** → your main component
* **`index.html`** → root HTML file

---

# 🔥 Quick sanity check

Edit:

```jsx
src/App.jsx
```

Change something like:

```jsx
<h1>Hello React</h1>
```

→ Browser auto-reloads instantly ⚡

---

# ❗ Prerequisite

Make sure you have:

* Node.js (v18+ recommended)

Check with:

```bash
node -v
```
